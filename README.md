### Hoop-Server

# 📖 같이 농구할 사람을 구하는 앱
## 프로젝트 설명
- 농구를 하고 싶어도 같이할 인원이 충분하지 못해 하지못하는 사람들은 위한 토이 프로젝트

1. 개발 환경
   
Front-End/App : iOS

BackEnd : Spring boot 3

배포 : AWS EC2

CICD : Jenkins

디자인 : Figma

티켓 : Jira
